# 🚀 Quick Start - Install Node.js and Run the Server

## The Problem
Node.js is not installed on your system. You need it to run the development server.

## The Solution (Choose One)

### ⚡ Option 1: Quick Install (Recommended)
Download and install Node.js from the official website:

1. **Go to:** https://nodejs.org/
2. **Click:** "Download Node.js (LTS)" - the green button
3. **Install:** Double-click the downloaded file and follow the installer
4. **Restart Terminal:** Close and reopen your terminal
5. **Verify:**
   ```bash
   node --version
   npm --version
   ```
6. **Run the project:**
   ```bash
   cd /Users/meetmehta/edu-portal-gra
   npm install
   npm run dev
   ```

### 🔧 Option 2: Install via Homebrew
If you prefer using Homebrew:

```bash
# Install Homebrew (if needed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Node.js
brew install node

# Verify
node --version
npm --version

# Run the project
cd /Users/meetmehta/edu-portal-gra
npm install
npm run dev
```

## ✅ After Installation

Once Node.js is installed, run these commands:

```bash
# Navigate to project
cd /Users/meetmehta/edu-portal-gra

# Install dependencies (if not already installed)
npm install

# Start the server
npm run dev
```

Then open: **http://localhost:8080/use-case-finder**
